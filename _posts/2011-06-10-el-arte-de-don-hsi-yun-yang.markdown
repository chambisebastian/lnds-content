Prueben esto:


# wget [http://www.de.ioccc.org/2000/dhyang.c](http://www.de.ioccc.org/2000/dhyang.c)




# cc dhyang.c -o saitou




# ./saitou > aku.c




# cat aku.c




# cc aku.c -o soku




# ./soku > soku.c




# cat soku.c




# cc soku.c -o zan




# ./zan > zan.c




# cat zan.c




# cc zan.c -o aku




# ./aku | diff - aku.c
